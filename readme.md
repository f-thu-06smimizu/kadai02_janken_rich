## JS2 - じゃんけんアプリリッチver

## 制作アプリのタイトル　
運命の輪スロット ～黄金と剣の試練～

## 制作アプリの説明（40文字程度）
中世の古城を舞台にしたスロットゲーム。
騎士の栄光をつかめ！

## 工夫した点・こだわった点
- 中世の世界観の徹底
背景画像にを採用し、ゲームコンテナを中央上部に配置することで、背景画像の下部に描かれた豪華なロゴが見えるように調整。

- スロットリールの絵文字にCSSのフィルタ（grayscaleやsepia）を適用し、カラフルな絵文字を古びたブロンズのシンボルのように見せることで、世界観を統一。

- 可能な限り没入感を表現
結果表示のメッセージを「王権が、富をもたらす！」や「フォルトゥナは、微笑まず。」など、中世の運命論的な表現に変更。

勝利時に結果表示エリア（#result-display）の枠線や影を金色に強調し、大当たりを視覚的に表現。

- 情報の整理と利便性
配当表（プライズテーブル）をスロット本体の左側に配置し、中世の「巻物」のようなデザインにすることで、ゲームの情報が一目でわかるように工夫。

## 次回トライしたこと（または機能）
- サウンド要素の導入
ボタンクリック、スロット回転開始、勝利、敗北など、各アクションに合わせた中世風の効果音をJavaScriptのAudioオブジェクトを使って実装する。

- ボーナスゲームの追加
大当たり時に騎士の決闘などの簡単なミニゲームを発生させ、追加の配当を獲得できる要素を追加し、ゲーム性を深める。

- レスポンシブ対応
CSSの@mediaクエリを使用し、スマートフォンなどの小さな画面でも配当表とスロットが縦に並ぶようにレイアウトを調整する。

## 備考（感想、シェアしたいこと等なんでも）
背景画像の見切れやCSSのパスの問題など、予期せぬトラブルがありましたが、ひとつひとつ原因を特定し解決できたことで、CSSのパスやマージンの仕組みに丁寧に向き合う機会となりました。特にbackground-position: center bottom;やネガティブマージンを使ったレイアウト調整は、非常に勉強になりました。
